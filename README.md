Selamat datang di Tugas 1 Path Learning Divisi Backend!
Pada tugas perdana ini, kita akan menguji pemahaman logika dasar JavaScript yang sangat krusial dalam pembuatan server-side logic. Kamu ditugaskan untuk melengkapi sebuah fungsi kalkulator sederhana yang dapat melakukan operasi matematika dasar dan menangani error dengan tepat.

# Instruksi pengerjaan

Disediakan sebuah kerangka fungsi hitungKalkulator. Tugasmu adalah mengisi bagian // code here dengan logika berikut:
Buat pengecekan kondisi untuk parameter operator.

- Operator yang valid adalah +, -, \*, dan /.
- Lakukan operasi matematika antara angka1 dan angka2 sesuai dengan operator yang diberikan, lalu simpan hasilnya ke variabel hasil.
- Validasi Khusus: Jika operator adalah pembagian (/) dan angka2 bernilai 0, fungsi wajib langsung mengembalikan string: "Error: Tidak bisa membagi dengan angka 0!".
- Jika operasi berhasil, kembalikan (return) kalimat hasil akhir menggunakan Template Literals dengan format: "Hasil dari [angka1] [operator] [angka2] adalah [hasil]".
